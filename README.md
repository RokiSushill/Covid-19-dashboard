COVID‑19 Analytics Dashboard

Project Overview

This project presents a country-wise and date-wise COVID‑19 analytics dashboard. It focuses on tracking, reporting, and visualizing pandemic metrics to generate actionable insights for decision-makers.

Designed for recruiters and hiring managers, this dashboard demonstrates data cleaning, reporting, and visualization skills for real-world business analysis.

🔹 Key Metrics

Daily Metrics

New cases

New deaths

New vaccinations

Cumulative Metrics

Total cases

Total deaths

Total vaccinations

Derived Insights

Infection rate

Vaccination percentage

Country comparison trends

📊 Dashboard Features

Interactive Line Charts – Daily and cumulative trends of infections and vaccinations

Bar Charts – Compare total deaths and vaccination coverage across countries

Donut/Pie Charts – Visualize vaccination distribution and percentage coverage

Filters & Drill-downs – Analyze data by country, date range, and metric
